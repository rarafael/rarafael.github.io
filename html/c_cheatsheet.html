libc Headers:
header header description
stdio.h standard input/output; Includes FILE type definition, file interaction
functions and interactions with stdin/stdout/stderr
stdlib.h standard library; exit definitions and general use OS functions,
memory management functions and number-ascii translations
stdint.h standard integers; type definition for fixed bit-sized integers,
definition of values (max/min) for each type
math.h math library; defines a lot of math-related functions, from equations
to comparisons and defines several constants and max values
string.h pointer manipulation; defines several pointer/string manipulation
functions like memcpy and memset
unistd.h miscellaneous type definitions and macros, OS functions and utility
functions for argument checking and process manipulation

keywords:

Keyword Usage
[TYPE] [VARIABLE]Defines VARIABLE as TYPE
typedef [DEFINITION] [TYPE]Makes DEFINITION an usable keyword TYPE
struct [NAME] { [MEMBERS] }Create a structure NAME with MEMBERS that can be accessed with '.' or '->'
enum [NAME] { [MEMBERS] }Creates a default 0 indexed enum NAME with MEMBERS
union [NAME] { [MEMBERS] }Creates an union NAME with MEMBERS (unions only use one pointer to store data)
register [VARIABLE]Suggests the compiler to store VARIABLE in a register when compiling
volatile [VARIABLE]Suggests that VARIABLE can change before being accessed again I.E. a hardware device like a mouse
goto [SUBROUTINE]Jumps to SUBROUTINE
if( [CONDITION] ) { [STATEMENT] }If CONDITION is true, then proceed to execute STATEMENT
else { [STATEMENT] }Use after 'if', case the condition is false, execute STATEMENT
while( [CONDITION] ) { [STATEMENT] }While CONDITION is true, execute STATEMENT
breakStop executing loop
return [RET]Return RET to function
const [TYPE]Make TYPE non-modifiable
for([START];[CONDITION];[DO]) { [STATEMENT] }Execute START once, and while CONDITION is true proceed to execute DO then STATEMENT
sizeof [THING]Returns 'size_t' size of THING
restrict [TYPE]Suggests to the compiler TYPE won't be used by another declaration, produces optimizations
do { [STATEMENT] } while ( [CONDITION] )Executes STATEMENT once and then while CONDITION is true
switch( [VARIABLE] ) case [CONDITION]: [STATEMENT]Switch case with VARIABLE, proceed to execute STATEMENT if CONDITION is true
default: [STATEMENT]Switch case default, if no case is true proceed to execute STATEMENT
auto [TYPE]Defines that the current TYPE will only be used in this scope block (implicit)
extern [TYPE]Allows TYPE to be used outside of the scope block
static [TYPE]Makes TYPE visible outside of it's scope block
